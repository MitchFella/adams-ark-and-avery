# Adams Ark & Avery
Adams Ark & Avery is a Java-based web application developed as part of the **Certificate in Software Engineering (TUS)**.  

## Project Overview

The application allows customers to:
- Register and log in
- Create pet bookings with check-in and check-out dates
- Select basic services during the booking process
- Log out and re-authenticate successfully

## Implemented Features

- User registration and login
- Session-based authentication
- Customer logout functionality
- Pet booking creation
- Basic service selection during booking
- Input validation (dates, required fields)
- Manual and automated testing
- 91% automated code coverage

## Technologies Used

### Backend
- Java (JDK 17)
- JavaServer Faces (JSF)
- Maven (dependency management and build)

### Testing
- JUnit for unit testing
- EclEmma for code coverage analysis (91%)

### Server
- Apache Tomcat

### Version Control
- Git
- GitHub
